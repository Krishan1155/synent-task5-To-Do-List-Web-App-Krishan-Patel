# To-Do List Web App

## Overview

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript. The application allows users to manage their daily tasks efficiently with support for adding, editing, deleting, and storing tasks using browser Local Storage.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Data persistence using Local Storage
- Responsive user interface
- User-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Project Structure

├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Clone the repository

git clone https://github.com/Krishan1155/synent-task5-To-Do-List-Web-App-Krishan-Patel.git

2. Open the project folder

cd todo-list-web-app

3. Open index.html in your browser

No additional installation is required.

## Functionality

### Add Task
Users can add a new task by entering text and clicking the Add button.

### Edit Task
Existing tasks can be modified using the Edit button.

### Delete Task
Tasks can be removed permanently using the Remove button.

### Local Storage
All tasks are automatically saved in browser Local Storage and remain available after page refresh.
